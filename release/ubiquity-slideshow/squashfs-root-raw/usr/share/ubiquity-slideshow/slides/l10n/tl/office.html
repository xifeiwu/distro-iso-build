Maging produktibo

Gamitin ang LibreOffice para gumawa ng propesyunal ng mga dokyumento, mga spreadsheet at mga presentasyon na angkop-na-angkop para sa Microsoft Office. Mag-archive ng mga dokyumento, mga email, o mga web page sa PDF. Magpadala at tumanggap ng mga file gamit ang Giver sa local network. Ipagamit sa iba ang mga printer o i-access ito gamit ang ibang computer.
